# Maze Algorithms Project

This project demonstrates **maze generation** and **maze solving (pathfinding)** algorithms implemented in Python.

## Features
- Random maze generation (randomized DFS / recursive backtracking)
- Maze solving using BFS (shortest path)

## Project Structure
```
maze-algorithms-project/
  main.py
  requirements.txt
  README.md
  src/
    maze.py
    solver.py
    utils.py
```

## How to Run
```bash
python main.py
```
